# 树莓派不吃灰 / Pi in Use

《树莓派不吃灰 / Pi in Use》永久开源地址为 https://github.com/zhaoolee/pi ，将Linux融入生活，为喜欢折腾树莓派的人，提供打开思路的资源，寓教于乐，让树莓派造福人类。

《Pi in Use / 树莓派不吃灰》The permanent open source address is https://github.com/zhaoolee/pi. It integrates Linux into life, provides resources to open the mind for those who like to fiddle with Raspberry Pi, educates while entertaining, and makes Raspberry Pi benefit mankind.

Even if you don't have Raspberry Pi hardware, it won't affect your reading of this series. This series uses Ubuntu as the base image. You can choose to **install Ubuntu on an old computer**, or install a virtual machine on Windows or macOS system, run Ubuntu, and keep it powered on.

## 为什么要使用树莓派？树莓派对比Windows家用电脑有什么优势？/ Why use Raspberry Pi? What are the advantages of Raspberry Pi compared to Windows home computers?

- 树莓派使用ARM架构的处理器，功耗极低，即使要负责给外接硬盘供电，最高功耗也不会超过15W

- Raspberry Pi uses an ARM architecture processor, with extremely low power consumption. Even if it is responsible for powering the external hard drive, the maximum power consumption will not exceed 15W.

- 树莓派接口丰富，提供了2个USB 2.0接口, 2个USB 3.0接口，两个micro-HDMI接口，1个Type-C接口，1个极为先进的耳机接口， 一个网线接口，40个GPIO引脚；

- Raspberry Pi has a rich interface, providing 2 USB 2.0 ports, 2 USB 3.0 ports, two micro-HDMI ports, 1 Type-C port, 1 extremely advanced headphone port, one Ethernet port, 40 GPIO pins.

- 树莓派体积小巧，只有信用卡大小，放到桌面丝毫不占空间，换个透明壳子，是极好的桌面装饰品。

- Raspberry Pi is compact in size, only the size of a credit card. It takes up no space on the desk. With a transparent case, it makes a great desktop decoration.

- 树莓派运行Linux，可以常年不关机，适合做家用服务器，如果你有外网域名和固定IP， 用frp做一个内网穿透，树莓派即可变身为一台性能不错的云服务器。

- Raspberry Pi runs Linux and can be kept on all year round. It is suitable as a home server. If you have an external domain name and a fixed IP, you can use frp for intranet penetration, and Raspberry Pi can be transformed into a decent cloud server.

- 高情商：树莓派性价比高！低情商：我穷，而树莓派恰好便宜；树莓派基金会是个非盈利组织，目的是以低价硬件及自由软件促进学校的基本计算机科学教育，售价基本为成本价，最新的4B版本，起售价为35美元，顶配8GB内存版本，也只有75美元。

- High EQ: Raspberry Pi has high cost performance! Low EQ: I'm poor, and Raspberry Pi is just cheap. The Raspberry Pi Foundation is a non-profit organization, aiming to promote basic computer science education in schools through low-cost hardware and free software. The price is basically at cost. The latest 4B version starts at $35, and the top configuration with 8GB of memory is only $75.

- 树莓派有很多镜像可玩，除了官方的Raspberry Pi OS , 还可以刷入Ubuntu(Linux世界非常流行的桌面发行版)，Windows10（懂得都懂）, Arch(迷你的Linux操作系统), Kali(预装大量黑客工具的操作系统), OpenWRT(开源路由器操作系统)。

- Raspberry Pi has many playable images. In addition to the official Raspberry Pi OS, you can also flash Ubuntu (a very popular desktop distribution in the Linux world), Windows 10 (those who understand will understand), Arch (a mini Linux operating system), Kali (an operating system pre-installed with a large number of hacking tools), OpenWRT (open source router operating system).

- 树莓派刷镜像很方便，只需插拔替换SD卡，即快速体验多个系统，另外树莓派也支持U盘启动，如果你有N个U盘，或N张SD卡，刷系统会非常快乐。

- It's very convenient to flash images on Raspberry Pi. You just need to plug and unplug the SD card to quickly experience multiple systems. In addition, Raspberry Pi also supports USB boot. If you have N USB drives, or N SD cards, you will be very happy to flash the system.

- 树莓派社区成熟，软件生态好，开源社区极为活跃，来自世界各地的极客们，分享自己奇妙的开源项目，连微软都为树莓派定制了Win10镜像。

- The Raspberry Pi community is mature, with a good software ecosystem, and the open-source community is extremely active. Geeks from all over the world share their wonderful open-source projects, and even Microsoft has customized a Win10 image for Raspberry Pi.

- 树莓派能持续摇人入伙， 硬件设计优秀，稳定性好， 软件开源，系统持续更新， 可以给学生玩的编程电脑，信用卡大小，仅需35美金！

- Raspberry Pi can continuously attract people to join. It has excellent hardware design and good stability. The software is open-source, and the system is continuously updated. It can be a programming computer for students to play with. It's the size of a credit card and only costs $35!

## 安装的方法很简单, 直接运行以下命令即可

```
curl -fsSL https://openclaw.ai/install.sh | bash
```

## 使用mcp

在clawhub 中 skill是一等公民，我们可以通过skill来获取mcp的能力。

首先安装clawdhub

```
npm i -g clawdhub
```

然后通过 `openclawn tui` , 进入终端， 输入对话`请使用 clawhub 安装 https://www.clawhub.ai/steipete/mcporter`, 执行完成后，mcp就可用了
